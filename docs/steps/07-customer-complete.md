# Step 07: お客様側 - 注文完了

注文完了画面を実装する。

## ゴール

- 注文完了後にシンプルな確認画面が表示される
- 追加注文のフローが動作する

---

## タスク

### 7.1 注文完了ページ

- [ ] `src/app/order/[token]/complete/page.tsx` を作成
  - **静的ページ**（データ取得不要）
  - 表示内容:
    - 「注文を受け付けました」メッセージ
    - 「追加注文する」ボタン → `/order/[token]` にリダイレクト
    - 「お支払いはレジにてお願いいたします」テキスト
  - お客様向けカラーパレットを適用

### 7.2 追加注文フロー

- [ ] 「追加注文する」ボタンタップで `/order/[token]` に遷移
- [ ] カートは空の状態になっていること（注文確定時にクリア済み）

### 7.3 動作確認

- [ ] 注文確定後に完了画面が表示される
- [ ] 「追加注文する」でメニュー画面に戻る
- [ ] カートが空の状態でメニュー画面が表示される
